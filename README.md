# Playwright TodoMVC Automation

This project demonstrates UI test automation using Playwright and JavaScript.

## Features
- Page Object Model (POM)
- Screenshot on success and failure
- Playwright HTML reports
- Hooks (beforeEach / afterEach)
- Public TodoMVC application

## Tech Stack
- Playwright - Install from "https://playwright.dev/docs/intro"
- JavaScript 
- Node.js -Install from "https://nodejs.org/en/download"
- Use IDE (I have used VS code here "https://code.visualstudio.com/download")

## How to Run
npm install
npx playwright test
